# FRONT END PROJECT

The front end project uses React with TypeScript and Redux.

### How to run project

1. Step 1: yarn
2. Step 2: yarn start

### Logic

1. Data from API: `https://api.escuelajs.co/api/v1/products`
2. Product detail API: `https://api.escuelajs.co/api/v1/products/name`
